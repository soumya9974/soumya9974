## 🚀 Featured Projects

### 🌍 [TravelBuddy](https://github.com/soumya9974/travelbuddy) — Group Travel Planning App
> Real-time collaborative travel planning with group chat and role-based access control

A full-stack group travel app where users can create travel groups, communicate via live chat, and manage trip logistics together. Features WebSocket-powered real-time messaging with typing indicators and online presence, JWT authentication, and admin moderation controls.

**Stack:** `Spring Boot` `React` `TypeScript` `WebSocket (STOMP)` `Tailwind CSS` `MySQL` `Docker`

---

### 💰 [Expense Tracker](https://github.com/soumya9974/expense-tracker) — Personal Finance Dashboard
> Full-stack budgeting app with interactive charts and per-category budget tracking

Users can log expenses, create custom categories with colors, set monthly budgets, and visualize spending patterns through pie charts and bar graphs. Features secure auth via Supabase, Row Level Security for data isolation, and a responsive design that works across devices.

**Stack:** `React` `Supabase (PostgreSQL)` `Recharts` `Vite`

---

### 🐦 [TweetApp Backend](https://github.com/soumya9974/TweetAppBackend) — Micro Social Platform API
> Production-grade backend with observability and event streaming built in

A Spring Boot REST API for a Twitter-style short-post platform, integrated with an enterprise observability stack: Kafka for event streaming, Elasticsearch + Logstash + Kibana (ELK) for log aggregation, Prometheus + Grafana for metrics, and Swagger for API documentation.

**Stack:** `Spring Boot` `Java` `MySQL` `Apache Kafka` `Elasticsearch` `Prometheus` `Grafana`

---

### 🍽️ [MenuItem Management System](https://github.com/soumya9974/MenuItem) — RESTful Menu API
> Clean, layered Spring Boot service for managing restaurant menu items

A well-structured REST API following standard controller → service → repository architecture. Provides full CRUD for menu items with support for MySQL (production) and H2 (testing), easily composable into larger microservice ecosystems like restaurant or food ordering platforms.

**Stack:** `Spring Boot` `Java` `MySQL` `H2` `Maven`

### 🏦 Ocwen Loan Application *(Cognizant — Enterprise)*

Built and maintained as part of a professional team at Cognizant. Handled complex loan lifecycle management, integrating a Spring Boot backend with an Angular frontend over a MySQL data layer.
