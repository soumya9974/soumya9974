### 🌍 [TravelBuddy](https://github.com/soumya9974/travelbuddy) — Group Travel Planning App
> Real-time collaborative travel planning with group chat and role-based access control

A full-stack application where users create travel groups, communicate via live chat, and coordinate trips together. Built with a layered Spring Boot backend (controller → service → repository), JWT authentication, and WebSocket-powered real-time messaging with typing indicators and admin moderation. Containerised with Docker and deployed on AWS.

**Stack:** `Java` `Spring Boot` `JWT` `WebSocket (STOMP)` `React` `TypeScript` `Tailwind CSS` `MySQL` `Docker` `AWS`

### 🐦 [TweetApp Backend](https://github.com/soumya9974/TweetAppBackend) — Social Platform API with Full Observability
> Production-grade Spring Boot API with event streaming and an enterprise monitoring stack

A tweet management REST API integrated with a full observability pipeline: Apache Kafka for event streaming, ELK stack (Elasticsearch, Logstash, Kibana) for log aggregation, and Prometheus + Grafana for live metrics. API documented via Swagger and health-monitored via Spring Actuator. Containerised with Docker; explored Kubernetes for orchestration.

**Stack:** `Java` `Spring Boot` `MongoDB` `Angular` `Apache Kafka` `Elasticsearch` `Prometheus` `Grafana` `Docker`

### 💰 [Expense Tracker](https://github.com/soumya9974/expense-tracker) — Personal Finance Dashboard
> React + Supabase finance app with budget tracking and interactive data visualisation

A personal budgeting tool with custom expense categories, monthly budget limits, and a visual analytics dashboard (pie charts, bar graphs, spending trends). Features Supabase authentication and Row Level Security to isolate per-user data. Built using React and Recharts on the frontend, with Supabase/PostgreSQL as the backend. AI-assisted development tools (GitHub Copilot, Bolt AI) were used for prototyping and accelerating delivery.

**Stack:** `React` `Supabase (PostgreSQL)` `Recharts` `Vite` `Tailwind CSS`
